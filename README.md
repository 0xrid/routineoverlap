# Overlap

**Overlap** is a lightweight, mobile-first scheduling web application designed to help teams or groups visualize overlapping availability across a defined set of daily time slots.

## 🚀 Features

* **Customizable Roster**: Easily add, color-code, and toggle team members on and off.
* **Intuitive Grid**: A clear, color-coded visual matrix showing who is available at specific times.
* **Mobile-Optimized**: Designed with a compact, calendar-style interface that works seamlessly on phones and desktops.
* **Data Persistence**: Your schedule is automatically saved to your browser.
* **Import/Export**: Easily save your schedule to a JSON file or load one to share with others.
* **16:9 High-Fidelity Export**: Generate a clean, professional PNG image of your schedule for presentations, reports, or sharing in chat apps.

## 🛠 Tech Stack

* **HTML5 / Tailwind CSS**: For a responsive, modern UI.
* **Vanilla JavaScript**: No heavy frameworks—fast and lightweight.
* **html2canvas**: Used to capture the grid and convert it into a shareable high-quality image.

## 💻 How to Use

1. **Add People**: Open the menu (top-left) to add team members and assign them a unique color.
2. **Toggle Availability**: Clicking a person's name in the menu toggles them in the grid.
3. **Update Slots**: Click any cell in the grid to open a modal where you can assign specific people to specific time slots.
4. **Export/Save**:
* Use the **JSON** button to backup your data.
* Use the **Export 16:9** button to create a screenshot of your schedule for easy sharing.



## ⚙️ Deployment

This project is built as a single-file application (`index.html`). To host it:

1. Create a new GitHub repository.
2. Upload `index.html` to the root folder.
3. Enable **GitHub Pages** in the repository **Settings**.
4. Your app will be live at `https://[your-username].github.io/[repo-name]/`.
